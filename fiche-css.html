🎨 Fiche de Révision — CSS

Tout ce qu'il faut savoir pour l'examen — en français simple !

🎯 1. C'est quoi le CSS ?

Le CSS (Cascading Style Sheets) sert à mettre en forme les pages HTML : couleurs, tailles, positions, polices…

/* Syntaxe de base */ selecteur { propriété: valeur; propriété: valeur; } /* Exemple concret */ p { color: blue; font-size: 16px; }
💡 On écrit le CSS dans un fichier .css séparé, qu'on relie au HTML avec <link rel="stylesheet" href="style.css" /> dans le <head>.

📦 4. Le modèle de boîte (Box Model)

Chaque élément HTML est une boîte avec 4 couches :

🟡 MARGIN — espace DEHORS (entre les éléments)
🔴 BORDER — la bordure
🟢 PADDING — espace DEDANS (entre contenu et bordure)
CONTENU (texte, image...)
.boite { width: 200px; ← largeur du contenu padding: 16px; ← espace intérieur border: 2px solid black; ← bordure margin: 10px; ← espace extérieur }
💡 box-sizing: border-box — avec cette propriété, la width inclut padding + border. Sans elle, la boîte est plus grande que prévu !

📐 6. Flexbox — aligner les éléments

Flexbox sert à aligner des éléments en ligne ou en colonne facilement.

.container { display: flex; ← active flexbox flex-direction: row; ← en ligne (défaut) | column = en colonne justify-content: center; ← aligne sur l'axe PRINCIPAL align-items: center; ← aligne sur l'axe SECONDAIRE gap: 16px; ← espace entre les enfants flex-wrap: wrap; ← passe à la ligne si pas assez de place }

justify-content (axe principal)

flex-startDébut
flex-endFin

space-betweenEspacé (bords inclus)
space-aroundEspacé (avec marges)

align-items (axe secondaire)

flex-startHaut
flex-endBas

⚠️ Affirmation FAUSSE à l'exam : "En flex, justify-content aligne horizontalement et align-items verticalement" → C'est FAUX ! Ça dépend de flex-direction. Si direction = column, c'est l'inverse !

🔲 7. Grid — grille de mise en page

Grid sert à organiser des éléments en lignes ET colonnes en même temps.

.grille { display: grid; grid-template-columns: 1fr 1fr 1fr; ← 3 colonnes égales grid-template-rows: auto; gap: 20px; ← espace entre les cases }
✅ Grid simplifie le responsive design : on peut changer le nombre de colonnes selon la taille de l'écran avec @media.

📍 8. La propriété position

ValeurCe que ça faitPar rapport à quoi ?
staticPosition normale (défaut)Le flux du document
relativeSe déplace par rapport à sa position normaleLui-même
absoluteSort du flux, se positionne par rapport au premier parent non-static⭐ Premier parent non-static
fixedReste fixe quand on scrolleLa fenêtre du navigateur
stickyNormal jusqu'à un seuil, puis fixeLa fenêtre (après le seuil)
.parent { position: relative; } ← référence pour l'enfant absolute .enfant { position: absolute; top: 10px; right: 10px; }
💡 position: absolute = se positionne par rapport au premier parent qui a une position autre que static. S'il n'y en a pas, c'est par rapport à la page entière.
